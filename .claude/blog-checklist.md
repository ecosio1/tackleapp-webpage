# Blog Post Quick Checklist

Use this before publishing any blog post.

---

## Structure (12 Required Sections)

- [ ] **Section 1**: Above fold (title + promise + "best for" + "do this first")
- [ ] **Section 2**: Quick answer (3-6 bullets, no fluff)
- [ ] **Section 3**: Tackle box snapshot (copyable setup)
- [ ] **Section 4**: Step-by-step (5 numbered steps)
- [ ] **Section 5**: Decision tree (if/then conditions)
- [ ] **Section 6**: Spot playbook (structure + where fish stage)
- [ ] **Section 7**: Mistakes that kill the bite (5-10 bullets)
- [ ] **Section 8**: App CTA #1 (after top content)
- [ ] **Section 9**: FAQs (5+ questions)
- [ ] **Section 10**: 1-minute action plan (tight checklist)
- [ ] **Section 11**: Next step links (3-5 contextual)
- [ ] **Section 12**: Regulations line (neutral only)

---

## Images (4 MINIMUM Required)

- [ ] **Hero image** (1200x600, fishing action/location) - auto-rendered at top
- [ ] **Inline image #1** - After Section 3 (Tackle Box) - gear/lures laid out
- [ ] **Inline image #2** - After Section 4 (Step-by-Step) - casting/technique
- [ ] **Inline image #3** - After Section 6 (Spot Playbook) - structure/habitat
- [ ] **Inline image #4** (optional) - Before Section 7 (Mistakes) - fish/detail closeup
- [ ] All images have descriptive alt text (species + location + action)
- [ ] All images have actionable captions (what + why + when/how)
- [ ] Using markdown image syntax with Unsplash URLs (?w=800&h=600&fit=crop)

---

## SEO Metadata

- [ ] Title: 50-60 chars with primary keyword + location
- [ ] Meta description: 140-155 chars with action + benefit
- [ ] Primary keyword in first H2
- [ ] 3-5 contextual internal links
- [ ] Descriptive anchor text (not "click here")

---

## Content Quality

- [ ] 1500-2500 words total
- [ ] No paragraph longer than 5 lines
- [ ] Specific details (depths, weights, colors, times)
- [ ] Conditions included (water clarity, tide, wind)
- [ ] Common mistakes addressed
- [ ] No generic AI phrases ("dive into", "unlock")
- [ ] No regulatory claims or seasons/limits

---

## Conversion Elements

- [ ] 2 App CTAs (one top, one bottom)
- [ ] CTAs feel helpful, not salesy
- [ ] Clear next action for reader
- [ ] Related content cards with images

---

## Before Publishing

- [ ] Mobile preview looks good
- [ ] All links work
- [ ] Images load properly
- [ ] No TypeScript errors
- [ ] Reading time accurate
- [ ] Category assigned correctly
- [ ] Author attribution correct
- [ ] Published date set

---

## Red Flags to Avoid

❌ Long intro before quick answer
❌ Walls of text (no images)
❌ Vague advice without conditions
❌ More than 5 internal links (feels spammy)
❌ Marketing speak instead of practical tips
❌ Missing "do this first" callout
❌ No decision tree or mistakes section
❌ Regulatory advice

---

## Success Metrics to Check After 7 Days

- [ ] Average time on page: 3+ minutes
- [ ] Scroll depth to 75%: >40%
- [ ] Internal link clicks: 1.5+ per session
- [ ] Bounce rate: <60%
- [ ] CTA click rate: Track in analytics
